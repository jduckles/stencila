## Testing locally

In order to test the extension in VSCode, you will need to do the following:

1. run `npm install`
2. run `npm run compile`
3. open a `.smd` file (e.g. `src/fixtures/syntax.smd`) & hit `F5` to open a
   debug session with the extension installed.
